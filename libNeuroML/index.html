libNeuroML

Python libNeuroML is an API for working with neuronal models specified in NeuroML (http://neuroml.org)

NeuroML provides an object model for describing neuronal morphologies, ion channels, synapses and 3D network structure.

This package provides a Python API for working with simulator-independent representations of biophysically- and morphologically-detailed neuron models, with an efficient internal representation and serialization to/from NeuroML.
